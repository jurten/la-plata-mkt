# Marketing Agency Website Project Plan

## Project Overview
A high-end, modern website for a marketing agency, focusing on a clean, "Studio Fables" inspired aesthetic. The site will feature modern animations, language switching (English/Spanish), and a light/dark mode toggle.

## Proposed Screens

1. **Homepage (Landing)**
   *   High-impact hero section with kinetic typography animations.
   *   Featured projects carousel.
   *   Brief agency philosophy section.
   *   Global navigation with language (ENG/ESP) and theme (Light/Dark) toggles.

2. **Work / Portfolio**
   *   A grid-based showcase of agency projects.
   *   Hover animations for project thumbnails.
   *   Filterable by service type.

3. **Services**
   *   Detailed breakdown of agency offerings (Strategy, Creative, Digital).
   *   Clean, icon-driven layout with smooth scroll transitions.

4. **About Us**
   *   Agency story and team highlight.
   *   Focus on "minimalist sophistication" imagery.

5. **Contact / Let's Talk**
   *   Minimalist inquiry form.
   *   Office locations and contact details.

## Technical & Design Details
*   **Design System:** Fables Agency (Kinetic Minimalist).
*   **Animations:** Using smooth transitions, staggered entrance animations, and subtle hover effects to mimic the Studio Fables feel.
*   **Multi-language:** UI will include a clear ENG/ESP toggle in the header.
*   **Theme Switch:** A dedicated toggle to switch between light and dark modes seamlessly.
